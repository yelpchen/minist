# minist

这个项目用于学习 minist 的基础训练流程。

> 如果这里的 `minist` 指的是经典手写数字数据集 `MNIST`，后续可以把项目名和说明统一调整为 `mnist`，避免和数据集名称混淆。

## 学习目标

- 理解训练一个简单模型的基本步骤
- 熟悉数据加载、模型定义、训练循环和评估流程
- 通过小实验观察学习率、批大小、训练轮数等参数对结果的影响

## 项目结构

```text
.
├── README.md   # 项目说明和学习记录
└── test.py     # 后续可用于编写训练或实验代码
```

## 建议学习路线

1. 准备 Python 环境，并安装需要的机器学习库。
2. 加载数据集，了解输入数据和标签的形状。
3. 定义一个最小可运行的模型。
4. 编写训练循环，记录 loss 和 accuracy。
5. 在测试集上评估模型效果。
6. 修改超参数，比较不同实验结果。
